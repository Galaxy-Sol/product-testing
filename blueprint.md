# Blueprint: 다국어 지원 웹사이트

## 개요

이 프로젝트는 정적 웹사이트에 다국어 지원을 추가하는 것을 목표로 합니다. 사용자는 영어와 한국어 버전을 선택하여 볼 수 있습니다.

## 기능

*   **언어 선택:** 사용자는 각 페이지에 있는 링크를 통해 영어와 한국어 버전을 전환할 수 있습니다.
*   **정적 페이지:** 각 언어 버전은 별도의 HTML 파일로 제공됩니다.

## 이번 변경사항 계획

1.  **한국어 버전 생성:** `index.html` 파일을 복사하여 `index_ko.html` 파일을 생성하고 한국어로 번역합니다.
2.  **언어 전환 링크 추가:** 각 HTML 파일에 다른 언어 버전으로 이동할 수 있는 링크를 추가합니다.
3.  **GitHub 배포:** 변경된 코드를 GitHub 저장소에 푸시하여 배포합니다.
